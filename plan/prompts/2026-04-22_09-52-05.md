# Increment 9 Prompt

Task: Implement Increment 9 for this slot machine project.

Objective:
Integrate a legal compliance footer containing required disclaimers and hotline information, ensuring it is visible, readable, and does not disrupt gameplay or UI usability.

Requirements:

1. Add a footer containing:
   - "For amusement only"
   - "No real money prizes"
   - "21+"
   - California problem gambling hotline

2. Ensure visibility:
   - footer must be present in the main UI
   - must not be hidden or obscured

3. Ensure readability:
   - text must be legible and appropriately styled
   - must not blend into background

4. Integrate cleanly:
   - footer must fit within the existing layout
   - must not overlap or break existing UI

5. Preserve usability:
   - controls must remain accessible
   - gameplay must not be impacted

6. Handle layout edge cases:
   - ensure footer works on typical screen sizes
   - avoid pushing critical UI elements off-screen

7. Keep implementation simple and maintainable.

8. Keep changes scoped to footer integration only.

9. Do not break existing tests or linting.

Expected deliverables:

1. Footer added to application UI
2. Supporting styling adjustments
3. Existing app still runs correctly end-to-end
4. A short summary stating:
   - what content was added
   - how it was integrated
   - which files were modified

Constraints:

- Do not change gameplay logic
- Do not add popups or overlays
- Do not introduce dark patterns
- Do not redesign UI
- Do not introduce unrelated refactors

Success condition:
A visible, readable legal footer is integrated into the UI without affecting usability or gameplay.
