# Increment 14.5 Prompt: Splash Screen Button Overhaul

Refactor the Splash Screen (Main Menu) UI to elevate the "Social" and "Paytable" features to primary navigation buttons, matching the visual weight of the "Enter Casino" button.

## Requirements

1. **Button Layout (index.html):**
   - Locate the `#main-menu` overlay.
   - Add two new large buttons: **"PAYTABLE"** and **"SOCIAL"**.
   - **Placement:** Position these buttons prominently below or alongside the "ENTER CASINO" button to create a professional "Three-Button" home screen layout.

2. **Styling (style.css):**
   - Apply the same "Big Button" styling used for the Start button to these new entries (Gold gradients, 3D shadows, bold text).
   - Ensure the layout remains responsive on mobile (buttons should likely stack vertically or form a grid).

3. **Logic Integration (view.js):**
   - Attach click listeners to these new buttons.
   - **Behavior:** Clicking "PAYTABLE" or "SOCIAL" from the main menu should:
     1. Open the side drawer (`#side-drawer`).
     2. Automatically switch to the corresponding tab (Paytable or Social).
     3. **Note:** This should NOT hide the main menu yet; the user should be able to view the info and then close the drawer to return to the Splash Screen.

4. **Refining the "Start" Flow:**
   - Keep the "ENTER CASINO" button as the primary action that dismisses the menu and unlocks the audio.

## Deliverables
- Updated `#main-menu` structure in `index.html`.
- CSS updates for a multi-button splash screen layout.
- Logic in `view.js` to handle direct-to-tab drawer opening from the menu.

## Constraints
- Do not change the underlying `GameManager` logic or `Wallet` state.
- Maintain the "Vegas Gold & Purple" aesthetic.

## Success Condition
The Splash Screen no longer looks empty. It features a professional suite of large buttons that allow the user to explore the game's social features and rules before they start spinning.
