# Increment 14 Prompt: Social Hub & Functional Settings Modal

Expand the Main Menu and UI to include a functional Settings menu and a "Social Hub" to mimic the high-engagement UX of mobile gambling apps.

## Requirements

1. **The Sidebar/Drawer System:**
   - Implement a "Slide-out Drawer" (`id="side-drawer"`) that can be toggled by the Settings button.
   - The drawer should contain tabs or sections for **Settings**, **Paytable**, and **Social**.

2. **Functional Settings:**
   - **Sound Toggle:** A checkbox or switch to mute/unmute the `AudioManager`.
   - **Volume Slider:** A range input that calls `AudioManager.setVolume()` to adjust the master gain.
   - **Reset Balance:** A "Emergency Funds" button that resets the `Wallet` to 1000 credits (for demo purposes).

3. **Social Hub (Mobile UX Mimicry):**
   - **Profile Section:** Display a placeholder avatar and a "Level 1 - Novice" badge.
   - **Friends List:** A scrollable list of 3-4 placeholder "Online Friends" with generic names and "Currently Playing" statuses.
   - **Why:** This creates the "Vegas" atmosphere of a busy casino floor.

4. **The Paytable:**
   - Add a visual guide showing the symbol values (e.g., "3x Cherries = 10x Bet"). This is crucial for user trust and clarity.

5. **Code Integration:**
   - **View:** Add `toggleDrawer()` logic and event listeners for the new inputs.
   - **Manager:** Ensure the game "pauses" or dims when the drawer is open.
   - **Audio:** Wire the volume slider and mute toggle to the existing `AudioManager.js` methods.

## Deliverables
- Updated `index.html` with drawer structure.
- CSS for the side-drawer (using slide-in animations).
- Functional logic in `view.js` and `AudioManager.js` for volume/mute control.

## Constraints
- **Theme:** Must match the existing "Gold & Purple" luxury aesthetic.
- **Logic:** Do not change the core spin math.
- **Responsiveness:** The drawer should not block the entire screen on desktop but should be full-width on mobile.

## Success Condition
The "Settings" button is no longer a dead link. It opens a polished drawer where the user can actually adjust the procedural audio volume and see their "Friends" list.
