# Increment 13 Prompt: Main Menu & Splash Screen Overlay

Implement a professional "Splash Screen" or Main Menu overlay that appears before the game begins. This layer serves as the entry point for the user and ensures the Audio Engine is properly initialized.

## Requirements

1. **UI Implementation:**
   - Create a full-screen overlay (`div id="main-menu"`) that sits on top of the slot machine game container.
   - **Theme:** Use the established "Vegas Gold & Purple" color scheme.
   - **Elements:**
     - **Game Title:** A stylized, high-saturation header (e.g., "GEMINI GOLD SLOTS").
     - **Start Button:** A prominent, animated button labeled "START GAME" or "ENTER CASINO".
     - **Background:** A subtle gradient or a dimmed view of the slot machine behind the overlay.

2. **State Management & Transitions:**
   - On page load, the game state should be "Paused" or "Idle" with the menu visible.
   - **Transition:** When the user clicks "Start Game," the menu should fade out or slide away using a CSS transition, revealing the slot machine.
   - **Audio Hook:** The "Start Game" click must trigger the `AudioManager.unlockContext()` method (created in Increment 12) to activate audio playback for the session.

3. **Architecture Integration:**
   - Update `src/ui/view.js` to manage the visibility of the overlay.
   - Update `src/main.js` to ensure the game doesn't start the background "ambient" sounds until the menu is dismissed.
   - Update `src/controller/GameManager.js` if necessary to prevent any keyboard shortcuts (like Space to spin) from working while the menu is active.

4. **Settings Placeholder:**
   - Add a small "Settings" icon or text link on the menu that opens a placeholder for "Sound On/Off" and "Paytable" (to be implemented in a later increment).

## Deliverables

- Updated `index.html` with the menu structure.
- Updated `src/style.css` with the "Vegas" splash screen styling.
- Updated `src/ui/view.js` and `src/main.js` to handle the transition logic.

## Constraints

- **Responsiveness:** The splash screen must look good on both mobile and desktop.
- **Theme:** Maintain the luxury casino aesthetic.
- **Ethics:** Include the "For Amusement Only" and "21+" labels clearly on the splash screen as well.

## Success Condition

The game loads to a beautiful splash screen. Clicking "Start" smoothly transitions the user into the game, unlocks the audio engine, and begins the background ambient music.
