# Increment 2 Prompt

Task: Implement Increment 2 for this slot machine project.

Objective:
Set up a code-quality toolchain for JavaScript, CSS, and HTML, then fix any issues it reports in the existing codebase without changing runtime behavior.

Requirements:

1. Add and configure the following development tools:
   - ESLint for JavaScript
   - Prettier for formatting
   - stylelint for CSS
   - HTMLHint for HTML

2. Create all required configuration files for those tools.

3. Update package.json with npm scripts at minimum for:
   - lint
   - lint:fix

4. The lint scripts must cover the current source and test code, specifically:
   - src/
   - tests/

5. Run the configured tools against the existing project files and fix all reported linting / formatting issues that are safe to correct automatically or manually.

6. Do not change program behavior.
   - Do not alter game logic.
   - Do not alter payout logic.
   - Do not alter architecture beyond what is necessary for lint / formatting setup.
   - Do not refactor unrelated code.
   - Do not rename files unless required for tool compatibility.
   - Do not introduce new features.

7. Preserve the current working spin loop and current UI behavior.

8. Keep the changes limited to Increment 2 scope only.
   - No Wallet tests yet.
   - No controller/view refactor yet.
   - No new gameplay features.
   - No UI polish work unless a lint fix requires a formatting-only change.
